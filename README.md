# Enterprise Network Project

This repository contains the configuration files and Packet Tracer project for a simulated enterprise network setup. The project includes routers, switches, and VLAN configuration, as well as NAT and access control.

## Contents

* `R0.cfg` – Router R0 configuration
* `R1.cfg` – Router R1 configuration
* `Core switch.cfg` – Core switch configuration
* `S1.cfg` – Access switch configuration
* `S2.cfg` – Access switch configuration
* `S3.cfg` – Access switch configuration
* `S4.cfg` – Wireless switch configuration
* `Enterprise_Network` – Packet Tracer project file

## Features

* Router configuration with RIP routing protocol
* NAT configuration and access control lists
* VLAN segmentation and trunking
* Switch port configuration with access and trunk modes
* Integration of wireless VLAN for AP connections

## Usage

1. Open the `.ptk` file in Cisco Packet Tracer to view and simulate the network.
2. Use the `.cfg` files to apply configurations to individual devices manually if needed.
